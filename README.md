Demo project (THAT I DONT WANT TO CONTINUE) with Node.js, has log in screen for user, it takes an excel (.xlsx) file via drag and drop and turns it into visual data with charts, supported types of rows are just three: "amount", "date", "description"(which is used for categories). The data is stored individually for each user, the user password is hashed within the database and stored locally, the app runs locally but it can be changed to run on a server. (Dark mode not available)
